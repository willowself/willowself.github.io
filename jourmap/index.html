jourmap - A space pioneers indexer

jourmap is the most powerful Space Pioneers indexer out there. It can provide you with any resources you might need and can help you find your way out of bad situations, such as being stranded.

Capabilities

The jourmap database contains every single planet, moon, and star in Space Pioneers. This makes it the one and only SP indexer that has all unscrapable values catalogued.

It stems from x4index, which unlike this bot focused only on functionality, and not looks or ease of use.

Status

jourmap is actively updated.

Focus

Looks, appeal, and functionality.

Mapping

the j!map command is a powerful tool that allows one to create an accurate picture of an SP starmap while also being able to add more information to it via the use of jourmap's Parsing Language.

• zoom: number
Controls the map's zoom

• target: int/galaxy/list
Centers the camera around a star id or a galaxy (mw, lmc, smc, sdig) or if a two value list is provided, it will center on the x, y coordinates from it

• line: {int, int}
Draws a yellow line between two stars

• route: {int, int, number}
Pathfinds a route from the first star to the second, where each warp can't be longer than the third number (in parsecs)

• qualityOverride: int
Allows you to select a custom image quality. Default is 1000.

• starRadiusOverride: number
Changes the visual size of the stars on the image, in pixels

• showStarIds: bool/int
If a bool, then it will show a star id next to every drawn star. If an int, it will do the same but you get to pick the text size, in pt

• colorCode: bool
Determines if stars from different sectors will be colored differently

• full: bool
Overrides "target" and "zoom" variables, forcing the map to show every star in the database at once

• hideSmoke: bool
Hides the galactic smoke and glare

• hideGlare: bool
Hides the milky way nucleus glow

• TEST&SET: bool
Appends the parsed values of each other argument to the metadata output

Routing

jourmap includes a pathfinder that can help you find a way to any star from your star even if you don't have a hyperdrive. In SP, liquid fuel ships can fly up to 3.5 parsecs distance, which enables you to just type 3.5 into the route argument's last value and see if maybe you aren't stranded after all.

Searching

the j!search command is an incredibly powerful jourmap command which will evaluate your conditions and go through every single planet out there and check if it matches your criteria. This includes resources and moons, so you can find a perfect place for your nation in just a few seconds.

Safety

Unlike x4index or x4alliance, jourmap's database is read-only. This makes it completely invulnerable to vandalism, and enables private usage.

You can use jourmap in your DMs, and nobody will know what you're up to.
